#green-bit web application
A sample web application created using MEAN stack.

#Installing And Running

Installing npm modules<br/>
  - npm install<br/>

Running mongoDB<br/>
  - mongod<br/>

Starting server<br/>
  - node server<br/>

Open Browser<br/>
  - http://localhost:3000/signup<br/>
